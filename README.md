# Food-Delivery-Analytics
Performed data analysis on 30,000+ records using SQL (joins, subqueries, aggregations) to evaluate customer behavior and delivery performance.
🎯 Objective

The goal of this project is to:

Analyze food order patterns and peak demand times
Understand customer preferences and ordering behavior
Track delivery efficiency and delays
Monitor revenue and business performance
📊 Key Features / Dashboard Modules
Overview: Total orders, revenue, average delivery time
Customers: Customer segmentation, repeat vs new users
Restaurants: Top-performing restaurants, cuisine analysis
Orders: Order trends by time, location, and category
Delivery: Delivery time analysis, delay tracking
Finance: Revenue, profit, and cost insights
📈 Key Insights
Identified peak ordering hours and high-demand days
Found top-performing restaurants and cuisines
Analyzed customer retention and repeat orders
Highlighted delivery delays and improvement areas
🛠 Tools & Technologies
Power BI
Excel / CSV Dataset
Data Cleaning & Transformation (Power Query)
DAX (Data Analysis Expressions)
📂 Dataset Description

The dataset includes:

Order details (date, time, amount)
Customer information
Restaurant data
Delivery partner performance
🚀 Business Impact
Helps improve delivery efficiency
Supports better restaurant partnerships
Enables targeted marketing strategies
Improves customer satisfaction
